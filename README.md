# aws-web-app-deploy
IaaC using terraform modules to deploy Web App on AWS
Deploy a Scalable Web Stack on AWS.
This stack will be used to provision infrastructure for a simple, scalable web application.
The app will run on EC2 instances in an Auto Scaling Group (ASG) behind a Load
Balancer, use RDS for data, and store static assets in S3, distributed via CloudFront.
